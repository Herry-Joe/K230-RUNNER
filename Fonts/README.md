# Fonts

This folder contains display/font resources used by the K230 runner game.

Required by the game:

- lv_font_simsun_16_cjk.fnt

Other files are bundled sample fonts from the CanMV environment and are kept here so the project can run on the target device without manual setup.

If you want a smaller repository, you may keep only lv_font_simsun_16_cjk.fnt and adjust the font path in code accordingly.
